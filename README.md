# 📱 Sensor Dashboard App

Sensor Dashboard App is a Flutter-based mobile application developed as part of a Mobile Programming practical assignment. The application demonstrates the integration of smartphone sensors by displaying real-time data from the **Accelerometer**, **Gyroscope**, **GPS**, and **Camera Preview**. It also implements runtime permission handling to ensure secure access to device features.

## ✨ Features

* 📊 Real-time Accelerometer monitoring
* 🔄 Real-time Gyroscope monitoring
* 📍 GPS location (Latitude, Longitude, Altitude, Speed, Accuracy)
* 📷 Camera Preview with camera switching support
* 🔐 Runtime permission handling for Location and Camera
* 🎨 Clean dashboard interface using Flutter Material Design

## 🛠️ Built With

* Flutter
* Dart
* sensors_plus
* geolocator
* camera
* permission_handler
* intl

## 📚 Purpose

This project was developed to learn how to access and utilize smartphone sensors in Flutter applications while implementing clean project structure, state management, and permission handling.
